# AION Update 004

Adds the first functional AION CLI generators.

## Capabilities

- Generate a service module
- Generate an ADR
- Generate a workflow specification
- Generate a documentation page
- Validate names
- Refuse accidental overwrites unless `--force` is used
- Work with macOS Bash 3.2

## Suggested commit

```bash
git add .
git commit -m "feat(cli): add service, ADR, workflow and documentation generators"
git push
```
