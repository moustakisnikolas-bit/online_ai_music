# AION Update 008

Adds the first real ambient audio engine.

## Adds

- White, pink and brown noise generation
- Sine tone and layered-tone generation
- Fade-in and fade-out
- Deterministic random seeds
- Preset-based generation
- Ambient audio job payload extensions
- Worker support for new audio types
- Tests for noise, tone and presets
- Documentation for audio profiles and safety limits

## Suggested commit

```bash
git add .
git commit -m "feat(audio): add ambient noise, tone layers and presets"
git push
```
