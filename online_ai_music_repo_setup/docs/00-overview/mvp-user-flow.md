# AION MVP User Flow

## Input

The user chooses:

- content title;
- purpose;
- audio mode;
- frequency;
- duration;
- channels;
- output format;
- artwork preset.

## Processing

AION creates:

- original audio;
- metadata;
- artwork;
- optional video;
- catalog manifest;
- export ZIP.

## Output

The user can preview and download the generated assets.

## Current Boundary

The MVP ends at export-package creation.

Uploading to YouTube or music distributors remains a manual, reviewed action.
